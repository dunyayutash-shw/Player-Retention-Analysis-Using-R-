# Player-Retention-Analysis-Using-R-
An R-based data analysis project exploring factors that influence player retention in online games through data cleaning, exploratory analysis and visualizations

## Introduction
Player retention is a key performance indicator for online gaming companies because retaining existing players is often cost-effective than acquiring new ones. Analyzing player retention data using R to identify patterns in player behavior by examining factors associated with player Engagement "while the dataset does not contain direct Retention measures, Engagement level is used as a proxy indicator of player activity and continued participation".

## objective 
finding key factors effecting player engagement and making insights that can support stategies improving player engagement and long-term retention.
